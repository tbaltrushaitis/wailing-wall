## :runner: Usage ##

### 1. :briefcase: Download ###
```shell
$ git clone https://github.com/tbaltrushaitis/wailing-wall.git wailing-wall && cd wailing-wall
```

### 2. :wrench: Setup ###
```shell
$ make
```
